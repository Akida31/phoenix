# Phoenix
Phoenix is a small interpreted programming language developed by Akida.

The Interpreter of it is written in Rust.

# Grammar
The Grammar of Phoenix can be found in `src/interpreter/grammar.txt`

# Features
- Lexing
- Parsing to AST
- Interpreting
- Variables
